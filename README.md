## Description:
Dr Serpent üêç in an easy to use testing framework for Python focusing on Behaviour Driven Development and Test Driven Development processes.

## Example:

```python

def run():
    return 1 + 1

# test_file
def test_number():
    Expect(run(2)).to_equal(2)
```
